# Tic Tac Toe

This is a simple Tic Tac Toe game made with Flutter. It has 2 player and AI feature. The AI is made with the help of Minimax and Alpha-Beta pruning algorithm.

## 📦 Installation

### Prerequisites

-   Flutter
-   Android Studio / Xcode

### Setup

1. Clone the repo

```sh
git clone
```

2. Install dependencies

```sh
dart pub get
```

3. Run the app

```sh
flutter run
```
